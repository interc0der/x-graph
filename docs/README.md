## File Structure

The file structure has been modified to match a conventional ReactJS project.

```
├── build
│   ├── chrome
│   ├── firefox
│   ├── opera
├── config
├── public
│   ├── index.html
│   ├── popup.html
│   ├── options.html
│   ├── manifest.json
├── scripts
├── src
│   ├── components (shared)
│   ├── pages
│   │   ├──  index
│   │   │   ├──   index.ts
│   │   │   ├──   index.scss
│   │   ├──  popup
│   │   │   ├──   index.ts
│   │   │   ├──   index.scss
│   │   ├──  options
│   │   │   ├──   index.ts
│   │   │   ├──   index.scss
├── docs
├── node_modules
├── jest.config.js
├── tsconfig.json
├── yarn.lock
├── package.json
```
